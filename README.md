# 好压 (LocalPress)

一款专注于 PDF 与图片文件体积优化的在线压缩工具，永久免费使用，无需注册即可体验。
✨ 核心功能
PDF 压缩：无损压缩 PDF 文件，显著减小体积，方便分享与存储
图片压缩：支持 JPG/PNG/WebP 等常见格式，可自定义压缩质量
批量处理：一次上传多个文件，同步完成压缩，提升效率
安全隐私：文件上传后自动清理，保障用户数据安全
完全免费：无使用次数限制、无文件大小门槛，开箱即用
🚀 快速开始
访问网站：好压（示例链接，替换为你的实际域名）
上传需要压缩的 PDF 或图片文件
选择压缩模式（默认推荐「均衡压缩」）
点击「开始压缩」，等待处理完成
下载压缩后的文件，直接使用
🛠️ 技术栈
框架：Next.js + TypeScript
样式：Tailwind CSS
部署：Vercel
核心压缩能力：基于浏览器端高效算法实现，无需后端依赖
📝 项目说明
好压致力于为个人用户提供轻量化、无负担的文件压缩服务，所有压缩操作均在浏览器端完成，不会上传你的文件到服务器，兼顾速度与隐私。
